## Max's Puzzles - Project Summary

Max's Puzzles is an educational maths puzzle platform I built as a gift for my 6-year-old. The first module, Circuit Challenge, is a path-finding game on a hexagonal grid where players solve arithmetic problems to navigate from start to finish. It features 10 difficulty levels spanning ages 4-11, a story mode with 10 chapters each guided by a unique alien character, a classroom-ready puzzle printer, and a parent dashboard for tracking progress. The whole thing is designed to make mental arithmetic genuinely fun - kids are solving dozens of calculations per game without it ever feeling like homework.

The platform ships as both a responsive web app and a native iOS app. The web stack is React 18 with TypeScript, Tailwind CSS, and Vite, using Supabase for authentication and family accounts, with IndexedDB for offline-first guest play. The iOS version is built in SwiftUI targeting iOS 16+, with CoreHaptics for tactile feedback and AVFoundation for background music. Both platforms share the same puzzle generation algorithm - a "connectors first" approach that guarantees every puzzle is valid and solvable in under 200ms. The architecture uses a hub-and-spoke pattern so new puzzle modules can be added without touching existing code.
